Project Title: Attendance System

Author: Zeyad Mohamed

Event: BEEST Event

How to use the website: It's very easy just open the open site and write the student name and phone number if you want then press absent or present.

I made this website because when the teachers are taking the attendance i feel very bored because they're using old ways


Here's some cool Features:
1- Student attendance recording
2- Present and absent status marking
3- Optional phone number support to in case of you want to contact with him
4- Attendance history table
5- Automatic attendance counting to make it much easier
6- Time tracking for each record to make sure that everyone is one time
