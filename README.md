## Why I made it:
I made this website because when the teachers are taking the attendance i feel very bored because they're using old ways


## Here's some cool Features:
- Student attendance recording
- Present and absent status marking
- Optional phone number support to in case of you want to contact with him
- Attendance history table
- Automatic attendance counting to make it much easier
- Time tracking for each record to make sure that everyone is one time
